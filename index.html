VHDL Viva Questions & Answers

1. What is the Full Form of VHDL? What is its use? ❓
VHDL stands for VHSIC Hardware Description Language. It is used to model and simulate digital systems before physical implementation.
2. What do you actually mean by very high speed IC? ⚡
VHSIC stands for Very High Speed Integrated Circuit, which refers to highly complex and high-performance ICs used in advanced digital systems.
3. Define EDA. What are the steps in EDA? 🛠️
EDA stands for Electronic Design Automation. Steps include:
- Design Entry
- Synthesis
- Simulation
- Implementation
- Verification
4. Name the other HDL language apart from VHDL. 🔥
Verilog is another hardware description language (HDL) commonly used in digital design.
5. Draw Y Diagram. What is its significance? 📚
The Y diagram represents three domains of digital system design:
- Behavioral Domain
- Structural Domain
- Physical Domain
6. What is library in VHDL? What library and logic type you have used in all VHDL programs? 📚
A library contains predefined packages and functions.
Common libraries:
- IEEE library
- STD_LOGIC_1164 package for logic types
- numeric_std for arithmetic operations
7. What is a module in VHDL? Write the module of a 4-bit counter/4-bit adder/4-bit Register/4-bit ALU. 🧩
A module in VHDL is described as an entity and its architecture. Example: 4-bit Counter
entity counter is
    Port ( clk, reset : in std_logic;
           q : out std_logic_vector(3 downto 0));
end counter;
architecture Behavioral of counter is
    signal temp : std_logic_vector(3 downto 0);
begin
    process (clk, reset)
    begin
        if reset = '1' then
            temp <= "0000";
        elsif rising_edge(clk) then
            temp <= temp + 1;
        end if;
    end process;
    q <= temp;
end Behavioral;
8. How many programming styles are supported in VHDL? 💡
Three styles:
- Dataflow Modeling
- Structural Modeling
- Behavioral Modeling
9. Define Dataflow, Structural and Behavioral Modelling. 🧠
- Dataflow Modeling: Uses concurrent signal assignments.
- Structural Modeling: Uses components and interconnections.
- Behavioral Modeling: Describes functionality using process blocks.
10. What do you mean by concurrent execution in VHDL? Write example. 🔁
Concurrent Execution: All statements outside processes run concurrently.
Example:
a <= b AND c;  -- Concurrent Assignment
d <= a OR b;   -- Concurrent Assignment
11. What is the difference between <= and := operators? ⚡
`<=` is used for signal assignment (concurrent), while `:=` is used for variable assignment (inside processes).
12. What is a process block in VHDL? 📦
A process block is a sequential block that executes when any signal in its sensitivity list changes.
13. What do you understand by sensitivity list? For a Flip-Flop design what are the tentative sensitivity list? ⏲️
A sensitivity list defines signals that trigger process execution. For a Flip-Flop: Sensitivity list includes `clk, reset`.
14. Explain a for loop to generate a square wave. Write the code. 🔄
process
begin
    for i in 0 to 7 loop
        signal_name <= not signal_name;
        wait for 10 ns;
    end loop;
end process;
15. What is the use of '&'? ➕
`&` is used for concatenation in VHDL.
16. How do you write a single bit and binary string like 10001101? 📝
Single bit: `'0'` or `'1'`
Binary string: `"10001101"`
17. How will you write the bus in VHDL? 🚌
signal bus : std_logic_vector(7 downto 0);
18. Define case structure in VHDL. 🧾
case sel is
    when "00" => y <= a;
    when "01" => y <= b;
    when others => y <= '0';
end case;
19. Write the if-else structure of JK Flip Flop. 🔄
process (clk)
begin
    if rising_edge(clk) then
        if reset = '1' then
            q <= '0';
        elsif (j = '1' and k = '0') then
            q <= '1';
        elsif (j = '0' and k = '1') then
            q <= '0';
        elsif (j = '1' and k = '1') then
            q <= not q;
        end if;
    end if;
end process;
20. Write the stimulus process of giving inputs to a Flip-Flop. ⏰
process
begin
    j <= '0'; k <= '0'; clk <= '0'; wait for 10 ns;
    j <= '1'; k <= '0'; wait for 10 ns;
    j <= '0'; k <= '1'; wait for 10 ns;
    j <= '1'; k <= '1'; wait for 10 ns;
